# Dentro do mecanismo: otimização de código com o Adobe Experience Manager Sites Optimizer

Descubra como a Adobe Experience Manager Sites Optimizer ajuda os desenvolvedores a aprimorar a qualidade do código detectando scripts redundantes e estruturas DOM pesadas. Esta sessão demonstra como usar a ferramenta para melhorar o Core Web Vitals, reduzir débitos técnicos e aumentar o desempenho. Saiba como aplicar recomendações facilmente no seu fluxo de trabalho para correções mais rápidas, SEO mais forte e experiências do usuário mais tranquilas.

Saiba como aplicar as recomendações do para reduzir débitos técnicos e melhorar o desempenho diretamente no AEM. Gravado ao vivo de San Jose.
