# Congelar, restaurar, inovar: Práticas com instantâneos do ambiente de desenvolvimento rápido para desenvolvimento na nuvem

Descubra como os ambientes de desenvolvimento rápido (RDEs) no Adobe Experience Manager Cloud Service permitem iterações nativas em nuvem rápidas. Saiba como os recursos de instantâneo permitem que os desenvolvedores congelem, restaurem e reutilizem estados completos de RDE. Isso permite simplificar os testes, reverter instantaneamente e acelerar os ciclos de desenvolvimento usando o plug-in RDE da CLI do Adobe I/O. Melhore a eficiência do desenvolvimento na nuvem e alinhe-se às políticas de governança corporativa.

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

Esta sessão — Congelar, Restaurar, Inovar: Práticas com Instantâneos do Ambiente de desenvolvimento rápido para desenvolvimento na nuvem — apresenta Remo Liechtenstein, Engenheiro de desenvolvimento de software da Adobe, e foi pré-gravada para o Adobe Developers Live 2025. Remo demonstra como os Ambientes de desenvolvimento rápido (RDEs) permitem uma iteração rápida e nativa em nuvem no Adobe Experience Manager Cloud Service e como novos recursos de instantâneo permitem que os desenvolvedores congelem, restaurem e reutilizem estados completos de RDE. Saiba como simplificar testes, reverter instantaneamente e acelerar ciclos de desenvolvimento na AEM Cloud usando o plug-in RDE da CLI do Adobe I/O.
