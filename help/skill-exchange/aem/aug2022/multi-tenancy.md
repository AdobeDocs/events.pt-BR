# Multilocação da sessão do AEM Champion

A sessão se concentrará em como vários locatários podem existir na mesma instância do AEM as a Cloud Service. O foco é o uso do Adobe Cloud Manager para implantar vários repositórios de código na mesma instância do AEM. Isso inclui a configuração de diferentes repositórios Git, bem como o uso de submódulos Git.
