# Introdução aos formulários Web no Acrobat Sign

>[!VIDEO](https://video.tv.adobe.com/v/3491645/?learn=on&enablevpops)

Ainda envia e-mails de um lado para o outro ou busca assinaturas de um contrato de cada vez? Os formulários web no Acrobat Sign permitem publicar um formulário assinado único que qualquer pessoa pode acessar, preencher e assinar — do seu site, um link compartilhado ou uma página incorporada — sem que você envie nada. Assista a este webinário prático, onde nossos especialistas abordam:

* O que são formulários web e onde eles brilham - Desde equipes de RH publicando pacotes de integração de nova contratação que centenas de funcionários podem concluir, até equipes financeiras postando aplicativos voltados para o cliente em um site para coletar assinaturas em escala e muito mais!
* Como criar e iniciar um Formulário Web - Assista à nossa apresentação em tempo real sobre como criar seu primeiro formulário Web, incluindo: carregar seu documento ou modelo, colocar campos de formulário, configurar a autenticação do assinante e publicar seu formulário como um URL compartilhável ou código incorporável.
* Como gerenciar e rastrear os resultados dos formulários Web — Monitore envios em tempo real, baixe dados de campos de formulário, edite ou desative um formulário em tempo real e mantenha seus formulários Web organizados à medida que o uso cresce.

Se você estiver coletando assinaturas de dez pessoas ou dez mil, deixará essa sessão pronta para criar, publicar e gerenciar seu primeiro formulário web com confiança.
