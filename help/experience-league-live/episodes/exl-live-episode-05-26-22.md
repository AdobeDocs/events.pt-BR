# Pergunte aos especialistas: noções básicas do Web SDK

Nesta primeira das três sessões de transmissão ao vivo sobre a coleção de dados do Adobe Experience Cloud, descubra sobre o &quot;futuro é agora&quot; da coleção de dados da Web com o Web SDK. Mostraremos todas as noções básicas do Web SDK, incluindo os motivos por trás, como ele funciona e os casos de uso importantes que ele possibilita. Teremos os especialistas conosco que o conhecem por dentro e por fora, para que possamos responder perguntas e mostrar as práticas recomendadas.
