# Vídeo de visão geral da versão 2025.08.0

Os recursos desta versão enfocam as inovações da Experience Manager Sites, Forms, Assets e Cloud Foundation e incluem o seguinte:

## Cloud Foundation

* Aumente a segurança e a confiança dos clientes fornecendo uma conexão segura e automação ao AEM CMK&#x200B;
* Produtos OIDC para publicação - Jira
* Traga seu próprio Git (BYOG)

## Formulários

* Eficiência do autor por meio de uma criação de regras mais fácil
* Botão OOTB para baixar DoR

## Sites

* Trazer Sua Própria Marca Para Cima
* Política de segurança de conteúdo
* Traga seu próprio Git
* Instantâneos e revisão
