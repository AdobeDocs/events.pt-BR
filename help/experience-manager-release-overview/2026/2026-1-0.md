# Vídeo de Visão geral da versão 2026.01.0 do AEM

Os recursos desta versão enfocam as inovações da Experience Manager Sites, Forms, Assets e Cloud Foundation e incluem o seguinte:

* AEM Sites
* Atualizações do Edge Delivery

* AEM Assets
* Pesquisa com IA (multilíngue, semântica)

* AEM Forms
* Interface de usuário associada de comunicações interativas

* AEM Cloud Foundation
* Servidor MCP
