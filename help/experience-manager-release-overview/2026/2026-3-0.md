# Vídeo de Visão geral da versão 2026.03.0 do AEM

>[!VIDEO](https://video.tv.adobe.com/v/3483065/?captions=por_br&learn=on&enablevpops)

Os recursos desta versão enfocam as inovações da Experience Manager Sites, Forms, Assets e Cloud Foundation e incluem o seguinte:

* Sites
   * Suporte a atualizações e inicializações do Experience Production Agent

* Formulários
   * Suporte de fluxo de trabalho para o editor IC
   * Suporte do Editor de temas para Componentes principais
   * Suporte ao controle de versão para Forms

* Ativos
   * AEM Content Advisor - GA com aplicativos Adobe
   * Conector do Microsoft Office (PowerPoint, Word, Excel)

* Cloud Foundation
   * Agente de governança: Gerenciamento de marca
   * Auditoria de permissões na interface de toque
   * Recurso de saída OIDC
   * Pausar Atualizações de Manutenção

* Cloud Manager
   * Cloud Manager MCP Server
