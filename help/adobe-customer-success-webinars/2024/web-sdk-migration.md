# Fundamentos para migração do Web SDK

Entenda as diferenças no aplicativo Measurement/AT.js vs. Web SDK, como migrar, considerações sobre o tempo de migração, opções de migração e diferenças de dados esperadas.

## Principais tópicos abordados

* O que é o Web SDK?
* Etapas e considerações da migração
* Tempo das migrações
* Erros e armadilhas comuns para evitar

>[!VIDEO](https://video.tv.adobe.com/v/3429291/?learn=on)


## Resumo

A reunião se concentrou nos aspectos essenciais da migração para a Web SDK, uma biblioteca da JavaScript que oferece benefícios para interagir com serviços de maneira agnóstica. &#x200B;Os principais pontos discutidos incluíram etapas para migração, como configurar permissões, configurar esquemas, criar fluxos de dados e mapear dados. Considerações foram feitas para lidar com variações de dados e determinar a ordem de migração dos clientes com o Analytics e o Target. Os insights foram compartilhados sobre métodos de rastreamento entre dispositivos, disparo de análises no início do carregamento da página e o significado da utilização de tags do Adobe. A reunião foi concluída com recomendações para concluir a pesquisa, registrar-se para os próximos webinários e expressar gratidão aos apresentadores e participantes por seu tempo e engajamento.
