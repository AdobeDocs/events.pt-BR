# Workfront Expert Insights: modo de texto com Katherine Lanning

Bem-vindo aos Insights de especialistas mais recentes!  Eu sou Cynthia Boon, sua CSM Workfront &quot;sob demanda&quot; e para este episódio temos o Workfront Text Mode Jedi, Katherine Lanning, Engenheira de soluções da Workfront da IQVIQ!  Durante este episódio, Katherine compartilha sua história, sua paixão por todas as coisas Text Mode Reporting, e demonstra como ela construiu seu painel de &quot;Gestão de Saúde&quot;!

O Modo de texto pode ser desafiador, mas as recompensas são incríveis.  Lembre-se do conselho da Katherine.

* &quot;Se você pode ver, você pode mudar isso.&quot; - Inverta para o modo texto em relatórios padrão para ver como as colunas são criadas.
* Aprenda fazendo retoques. : adicione campos, alterne para o modo texto e experimente para entender o que impulsiona os dados.
* Usar linguagem simples. - Coloque colunas de título como perguntas ou limpe declarações para que não administradores possam interpretá-las facilmente.
* Adicione descrições. - Personalize o texto em foco no modo de texto para explicar os dados e orientar os usuários.
* Torne as visualizações acessíveis. - Não dependa apenas de cores (por exemplo, vermelho/verde). Use ícones ou rótulos distintos.
* Aproveite a comunidade. - Conte com o Experience League, livros de receitas, API Explorer e colegas. Não reinvente a roda.
* Persista, mas tire férias. - Se travar, feche o notebook, dê uma caminhada e volte com os olhos frescos.
