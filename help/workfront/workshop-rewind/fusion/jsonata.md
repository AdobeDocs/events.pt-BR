# Criar JSON usando com o Workfront Fusion + JSONata

Hospedada em parceria com Steve Watts e Brian Haight do Adobe Professional Services, esta sessão abordou as noções básicas de leitura da notação de caminho JSON, compreensão dos tipos de dados e escolha do caminho inicial correto no Fusion.

Além disso, três exemplos reais foram abordados, desde localizar e filtrar dados até redefinir o JSON e fazer transformações avançadas em tempo real. É uma habilidade obrigatória para qualquer administrador do Fusion que deseja economizar tempo e simplificar o manuseio de dados!
