# Nathan&#39;s Five Faves - APIs básicas para o mundo da Workfront

Quer saber o que a API do Workfront pode realmente fazer por você? Junte-se a Nathan Johnson, Engenheiro de suporte técnico da Adobe III, para detalhar os fundamentos e trazer APIs para o mundo real.

Nesta sessão, Nathan aborda as noções básicas da API do Workfront, o que é, como funciona e por que é importante, antes de mergulhar em seus cinco casos de uso favoritos que mostram seu poder em ação.

>[!VIDEO](https://video.tv.adobe.com/v/3492587/?learn=on&enablevpops)

De conhecimentos fundamentais a aplicativos do mundo real, esta sessão fornecerá um caminho claro e acessível para o mundo das APIs do Workfront, guiado por um especialista que trabalha com elas todos os dias. Seja você novo nas APIs ou querendo expandir seu kit de ferramentas técnico, esta sessão ajudará a conectar os pontos entre conceito e capacidade.
